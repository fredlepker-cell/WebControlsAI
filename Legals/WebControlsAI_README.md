# WebControls

The WebControlsAI project provides VB code to produce a dll containing the DragList and MessageControl custom web controls

---

## 🚀 Features

- The Draglist control is a web list control that allows moving items within itself or for dragging and droping items to other web elements.
- The MessageControl is a message box based on the webforms message box.
